📦 Smart Order Optimizer (発注最適化アドオン)
需要予測システムから出力されたCSVデータを、サプライヤーごとの制約（MOQ・結束単位・休日）に合わせて自動調整し、実発注用データを生成するWebツールです。

🚀 ツールで解決できること
結束単位の自動補正: 2合、4合などのテープ括り単位に端数を自動切り上げ。

MOQ（最低発注量）達成の自動補完: 合計数量が不足している場合、需要日が近い商品から順に自動積み増し。

連休・休日前倒し: 特定の休日を指定することで、発注推奨日を自動で前営業日にスライド。

手動微調整: 自動計算された結果に対し、ブラウザ上で直感的に手動修正が可能。

🛠 使い方（クイックスタート）
設定の確認: 右側の「サプライヤー設定マスタ」に、各社のMOQや結束単位、休日リストをCSV形式で入力します。

データの読み込み: 需要予測システムから出力されたCSV（サプライヤー名, 商品名, 日付, 予測数, 結束単位）を選択します。

計算実行: 「⚡ 発注計算を実行」ボタンをクリックします。

内容の確認・修正: 各サプライヤーごとの判定結果（MOQ達成状況）を確認し、必要に応じて「発注数修正」欄の数字を書き換えます（リアルタイムに合計が再計算されます）。

ダウンロード: 「💾 統合CSVをダウンロード」をクリックし、タイムスタンプ付きの確定データを保存します。

🔒 セキュリティとプライバシー
データローカル処理: 本ツールは PyScript を使用しており、すべての計算はユーザーのブラウザ内で行われます。

外部送信なし: アップロードされたCSVデータやマスタ情報が外部サーバーに送信・保存されることはありません。機密性の高い需要データも安心して処理できます。
